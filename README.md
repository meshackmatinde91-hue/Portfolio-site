👟 ShoeStore – Modern E-Commerce Website

Welcome to the ShoeStore Website Project.

This project is a modern and responsive e-commerce landing page for a shoe store designed to showcase a stylish shoe collection and provide a smooth online shopping experience.

The website highlights featured products, seasonal promotions, and customer testimonials while maintaining a clean and modern user interface.

---

🚀 Project Overview

The website is designed to present a modern footwear store with a visually appealing layout and interactive elements.

Key sections included in the website:

- Hero section introducing the New Collection 2026
- Product categories such as Men, Women, Running, and Kids
- Featured product showcase
- Promotional Summer Sale banner
- Customer testimonials
- Newsletter subscription section
- Professional footer with social links

The goal of this project is to create a modern UI for an online shoe store that is responsive and user-friendly.

---

🎨 Design Features

- Clean and modern layout
- Responsive design for mobile, tablet, and desktop
- Smooth hover animations and transitions
- Product cards with pricing and action buttons
- Promotional sale banner for marketing campaigns
- Newsletter subscription section

The design focuses on simplicity, performance, and strong visual presentation of products.

---

🛠 Technologies Used

This project was built using modern web technologies:

- HTML5
- Tailwind CSS
- JavaScript
- Google Fonts (Inter)
- Lucide Icons

These technologies help create a fast, responsive, and visually appealing interface.
